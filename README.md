MedAI Pro is an AI-powered healthcare assistant built using **Streamlit**.  
It helps users analyze symptoms, get basic medical precautions, and identify which type of doctor to consult.

The application is designed to provide a simple and interactive healthcare support system for users.

---

## Features

- Symptom checker
- Doctor recommendations
- Medical precautions
- Login & Signup UI
- Attractive healthcare dashboard
- User-friendly interface

---

## Technologies Used

- Python  
- Streamlit  
- Pandas  
- NumPy  
- Scikit-learn  
- Plotly
